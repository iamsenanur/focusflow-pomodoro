# FocusFlow Pomodoro

A modern Pomodoro mobile app built with React Native and Expo.

## Features

- Custom focus timer
- Custom break timer
- Start / Pause / Reset controls
- Quick task selection
- Custom task input
- Focus / Break mode switching
- Completed task summary
- Celebration UI with sound, vibration, confetti, and dancing cat animation

## Tech Stack

- React Native
- Expo
- TypeScript
- React Hooks
- Expo Audio

## Demo Highlights

- Adjustable focus and break durations
- Task-based session tracking
- Case-insensitive task summary
- Clean modern mobile UI
